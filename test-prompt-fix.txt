ТЕСТ ИСПРАВЛЕНИЯ ПРОМПТА ДЛЯ ПОЛНОГО САЙТА
=============================================

ПРОБЛЕМА:
- В промпте для одного элемента "callout" есть поле ТИП_ВЫНОСКИ с подробными инструкциями
- В промпте для полного сайта этого поля НЕТ - только упрощённый формат

ИСПРАВЛЕНИЕ:
- Обновлена функция generateFullSitePromptWithElements()
- Обновлена функция generateOptimizedFullSitePromptWithElements()
- Теперь используется ПОЛНЫЙ промпт из библиотеки ELEMENT_PROMPTS для каждого выбранного элемента

ЧТО ИЗМЕНИЛОСЬ:
1. Вместо упрощённого формата берётся полный промпт из библиотеки
2. Извлекается раздел с форматом и инструкциями (включая ТИП_ВЫНОСКИ для callout)
3. Включаются все важные инструкции из оригинального промпта элемента

ОЖИДАЕМЫЙ РЕЗУЛЬТАТ:
- Теперь для callout в промпте полного сайта должно быть поле ТИП_ВЫНОСКИ
- Должны быть указаны допустимые категории: "Акция", "Новость", "Информация" и т.д.
- AI будет корректно генерировать выноски с правильным типом

ТЕСТ:
Выберите элемент "callout" в настройках промпта полного сайта и проверьте, что в сгенерированном промпте есть:
- ТИП_ВЫНОСКИ: [тип]
- Список допустимых категорий
- Подробные инструкции как в оригинальном промпте элемента
