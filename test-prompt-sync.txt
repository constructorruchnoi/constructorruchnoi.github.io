Тест синхронизации промптов между основной библиотекой и настройками полного сайта

1. Откройте AiParser в браузере
2. Сначала зайдите в основную библиотеку элементов и посмотрите промпт для "typography"
3. Затем зайдите в настройки полного сайта, выберите только раздел "О НАС" 
4. Выберите элемент "typography" в галочках
5. Нажмите "Применить настройки" в режиме "Полный"
6. Проверьте содержимое промпта - должны быть детальные инструкции из основной библиотеки

Ожидаемый результат:
- В промпте должна быть информация о том, что typography использует только СОДЕРЖИМОЕ без ЗАГОЛОВОК
- Должна быть ссылка на раздел ФОРМАТ из основного промпта
- НЕ должно быть упрощенных инструкций типа "Формат: ТИП: typography, ЗАГОЛОВОК: [заголовок]"

Проверить для элементов:
- typography
- callout  
- multiple-cards
- faq-section
- accordion
- testimonial-card

Все они должны содержать детальные инструкции из ELEMENT_PROMPTS, а не упрощенные.
